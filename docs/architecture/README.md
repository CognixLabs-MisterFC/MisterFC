# Architecture

Diagramas, modelos de datos y notas arquitecturales transversales que no encajan en una spec concreta ni en un ADR.

## Contenido esperado

- Diagrama de alto nivel del sistema (PWA + Supabase + Sentry + Vercel).
- Modelo de datos consolidado a medida que crece (E/R).
- Diagramas de flujo de procesos clave: convocatoria, toma de datos en partido, reporte mensual.
- Notas sobre integraciones, decisiones de naming, convenciones de tabla.

## Formato

- Diagramas en Mermaid (renderizable en GitHub) o exportados como PNG/SVG si Mermaid no llega.
- Markdown como contenedor principal.
