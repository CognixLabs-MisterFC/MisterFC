# Specs

Specs por subfase del Plan Maestro. Una spec por subfase no trivial.

## Cómo crear una nueva spec

1. Copia `_template.md` a `N.M-titulo.md` (ej. `2.4-vinculo-cuenta-padre-hijo.md`).
2. Rellena todas las secciones antes de empezar a implementar.
3. Mantén la spec actualizada durante la implementación si la realidad diverge del plan.
4. Al cerrar la subfase, la spec queda como artefacto histórico (no se borra).

## Naming

- `N.M-titulo.md` donde N.M es la subfase (ej. `1.3`, `7.2`).
- Título en kebab-case y descriptivo.
